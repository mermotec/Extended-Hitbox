## 🌌 **Overview**

<div align="center">

**Extended Hitbox** is a revolutionary client-side modification that redefines entity interaction in Minecraft.

Seamlessly expand hitboxes for **unparalleled targeting precision**, enhanced combat efficiency, and superior control—all without affecting server-side mechanics or other players' experiences.

</div>

### 🎯 **Why Extended Hitbox?**

<table>
<tr>
<td width="33%" align="center">

**⚡ Performance**

Zero lag, zero compromises. Optimized for lightning-fast execution.

</td>
<td width="33%" align="center">

Independent width/height control with real-time adjustments.

</td>

## 🎮 **Usage**

<div align="center">

### **🔥 Master the Controls**

</div>

### **⌨️ Keybinds**

<div align="center">

| Key | Action | Description |
|:---:|:------:|:------------|
| <kbd>H</kbd> | **Toggle Expansion** | Instantly enable/disable hitbox expansion |

*Default multiplier: **1.5x** — Fully customizable via commands*

</div>

<br>

### **💻 Command Reference**

<div align="center">

**All commands are client-side and begin with** `/hitbox`

</div>

```yaml
# Universal Control
/hitbox set <value>      # Set both width and height (0.1 - 10.0)

# Dimension-Specific
/hitbox width <value>    # Adjust width multiplier independently  
/hitbox height <value>   # Adjust height multiplier independently

# Management
/hitbox reset            # Restore defaults (1.0x) and disable
/hitbox get              # Display current configuration
```

<br>

### **📚 Examples**

<div align="center">

```bash
# Make all hitboxes 2.5x larger
/hitbox set 2.5

# Create wider targets for easier hits
/hitbox width 3.0

# Slightly taller hitboxes for aerial combat
/hitbox height 1.2

# Check current settings
/hitbox get
# Output: Hitbox: §a(ENABLED) | Width: 3.00x | Height: 1.20x

# Reset everything
/hitbox reset
```

## 🛠️ **Installation**

<div align="center">

### **🎯 Platform-Specific Guides**

</div>

<details>
<summary><b>📦 Fabric Installation</b></summary>

<br>

1. **Install Fabric Loader**
   - Download from [fabricmc.net](https://fabricmc.net/use/)
   - Select Minecraft 1.21.x
   - Run the installer

2. **Get Fabric API**
   - Download from [Modrinth](https://modrinth.com/mod/fabric-api) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
   - Version 0.119.4+ recommended

3. **Install Extended Hitbox**
   - Download from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
   - Place both `.jar` files in `.minecraft/mods`

4. **Launch Minecraft**
   - Select the Fabric profile
   - Enjoy!

</details>

<details>
<summary><b>🔧 NeoForge Installation</b></summary>

<br>

1. **Install NeoForge**
   - Download from [neoforged.net](https://neoforged.net/)
   - Select Minecraft 1.21.x
   - Run the installer

2. **Install Extended Hitbox**
   - Download from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
   - Place the `.jar` file in `.minecraft/mods`

3. **Launch Minecraft**
   - Select the NeoForge profile
   - Enjoy!

</details>

## 🔨 **Building from Source**

<div align="center">

### **👨‍💻 Developer Setup**

</div>

**Prerequisites:**
- Java 21 or higher
- Git

**Build Steps:**

```bash
# Clone the repository
git clone https://github.com/mermotec/Extended-Hitbox.git
cd Extended-Hitbox

# Build the project
./gradlew build          # Linux/macOS
gradlew.bat build        # Windows

# Find your compiled mod
cd build/libs
```

**Output:** `extendedhitbox-1.0.0.jar`
