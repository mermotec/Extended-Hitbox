### **💻 Command Reference**

<div align="center">

**All commands are client-side and begin with** `/hitbox`

</div>

```yaml
🎯 UNIVERSAL CONTROL
/hitbox set <value>      # Set both width and height (0.1 - 10.0)

📐 DIMENSION-SPECIFIC  
/hitbox width <value>    # Adjust width multiplier independently  
/hitbox height <value>   # Adjust height multiplier independently

⚙️ MANAGEMENT
/hitbox reset            # Restore defaults (1.0x) and disable
/hitbox get              # Display current configuration
```

<br>

### **📚 Examples**

<div align="center">

```bash
# 🔥 DOMINATE MODE - Massive hitboxes
/hitbox set 2.5

# 🎯 SNIPER MODE - Extra wide for ranged combat
/hitbox width 3.0

# 🚁 AERIAL MODE - Taller hitboxes for flying enemies
/hitbox height 1.2

# 📊 CHECK STATUS
/hitbox get
# Output: Hitbox: §a(ENABLED) | Width: 3.00x | Height: 1.20x

# 🔄 RESET TO NORMAL
/hitbox reset
```

## 🛠️ **INSTALLATION**

<img src="https://user-images.githubusercontent.com/74038190/212284136-03988914-d899-44b4-b1d9-4eeccf656e44.gif" width="50">

</div>

<br>

<details>
<summary><b>📦 FABRIC INSTALLATION</b></summary>

<br>

**Step 1: Install Fabric Loader**
- Download from [fabricmc.net](https://fabricmc.net/use/)
- Select your Minecraft version
- Run the installer

**Step 2: Get Fabric API**
- Download from [Modrinth](https://modrinth.com/mod/fabric-api) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
- Version 0.119.4+ recommended for 1.21.x

**Step 3: Install Extended Hitbox**
- Download from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
- Place both `.jar` files in `.minecraft/mods`

**Step 4: Launch & Enjoy!**
- Select the Fabric profile
- Press `H` to toggle hitbox expansion

</details>

<details>
<summary><b>🔧 NEOFORGE INSTALLATION</b></summary>

<br>

**Step 1: Install NeoForge**
- Download from [neoforged.net](https://neoforged.net/)
- Select your Minecraft version
- Run the installer

**Step 2: Install Extended Hitbox**
- Download from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
- Place the `.jar` file in `.minecraft/mods`

**Step 3: Launch & Enjoy!**
- Select the NeoForge profile
- Press `H` to toggle hitbox expansion

</details>
