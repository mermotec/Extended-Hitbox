## ✨ Features

**Extended Hitbox** transforms Minecraft combat and interaction through intelligent hitbox expansion.

- 🎯 **Dynamic Expansion** - All living entities (excludes player)
- ⚡ **Instant Control** - Toggle with `H` key, adjust on-the-fly
- 🎨 **Precision Tuning** - Independent width/height (0.1x - 10.0x)
- 💬 **Visual Feedback** - Color-coded action bar messages
- 🔧 **Powerful Commands** - Full `/hitbox` command suite
- 🚀 **Zero Performance Impact** - Optimized client-side execution

> **Multi-Platform:** Built for Fabric and NeoForge  
> ⚠️ **Forge discontinued** due to compatibility issues

---

## 🎮 Usage

### Keybind

| Key | Action |
|:---:|:-------|
| <kbd>H</kbd> | Toggle hitbox expansion (default: 1.5x) |

### Commands

```bash
/hitbox set <value>      # Set both dimensions (0.1 - 10.0)
/hitbox width <value>    # Adjust width only
/hitbox height <value>   # Adjust height only
/hitbox reset            # Restore defaults
/hitbox get              # Check current settings
```

**Examples:**
```bash
/hitbox set 2.5          # 2.5x larger hitboxes
/hitbox width 3.0        # 3x wider targets
/hitbox get              # View status
```

## 🛠️ Installation

<details>
<summary><b>📦 Fabric</b></summary>

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.x
2. Download [Fabric API](https://modrinth.com/mod/fabric-api) (v0.119.4+)
3. Download Extended Hitbox from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
4. Place both `.jar` files in `.minecraft/mods`
5. Launch and enjoy!

</details>

<details>
<summary><b>🔧 NeoForge</b></summary>

1. Install [NeoForge](https://neoforged.net/) for Minecraft 1.21.x
2. Download Extended Hitbox from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/extended-hitbox)
3. Place the `.jar` file in `.minecraft/mods`
4. Launch and enjoy!

</details>
